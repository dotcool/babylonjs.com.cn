Create Basic Elements (shapes/meshes)

In this tutorial, we are going to learn how to create basic elements using Babylon.js, such as boxes, spheres, and planes.

Playground Demo Scene 2 - Seven basic shapes/mesh

How can I do this ?

An easy way to get started using basic elements… is by visiting Playground Demo Scene 02. You might wish to use the ‘Get .zip’ choice on its top menu. The index.html that you will receive in that zip… contains everything you will need to start making basic elements. Remember that link, as we will talk more about it.

I’m sure you have already read the Babylon.js Primer and the previous tutorial, and thus you know how to format scene files. So, we won’t talk about that, here. We are going to go step by step through Playground Demo Scene 02. Open that link in a new tab or window, and then come back here, and we’ll get started.

Let’s look at the createScene function used in Playground Demo 2:

Starting with the box, we create 7 different basic elements, and then position them at the end of the function (to keep them from being atop one another). It’s really quite easy, isn’t it? Again, starting with the creation of the box, let’s talk about each of the 7 shapes/meshes.

  • Creation of a Box

Parameters are: name, size of the box, the scene to attach the mesh.

  • Creation of a Sphere

Parameters are: name, number of segments (highly detailed or not), size, scene to attach the mesh. Beware to adapt the number of segments to the size of your mesh ;)

  • Creation of a Plane

Parameters are: name, size, and scene to attach the mesh.

  • Creation of a Cylinder

Parameters are: name, height, diamTop, diamBottom, tessellation, [height subdivs], scene, updatable. The height subdivs parameter (shown set to 1) is optional.

  • Creation of a Torus

Parameters are: name, diameter, thickness, tessellation (highly detailed or not), scene, updatable.

  • Creation of a Knot

Parameters are: name, radius, tube, radialSegments, tubularSegments, p, q, scene, updatable
You can learn more about torus knots… RIGHT HERE.

  • Creation of a Lines Mesh

Parameters are: name, [array of comma-separated vectors], scene.

I could explain how the Lines Mesh constructor works, but I think you can see how it works just by looking at the demo code above. Notice the [ and ]. Those are the enclosing tokens for an array, yet another kind of Javascript value. The first vector3 of the array is the starting location for drawing lines. After that, a comma, and then the next vector3 location… indicating where the line is drawing-to next. Then, another comma, and another vector3 to a new location. You can add as many vectors as you wish, but notice that the LAST vector3 does not have a comma following it. Please make your array of vectors be formatted similarly.

More Basic Elements - Grounds

Up to this point, we have been talking about basic elements from our Playground Demo Scene 02, but three important mesh shapes (basic elements) are not included in that demo scene. They are each ways of making ‘ground’ in Babylon.js. Let’s take a look:

  • Creation of a Ground

Parameters are: name, width, depth, subdivs, scene

Our Playground Demo Scene 01 uses a CreateGround constructor… so you can see one in action by using the above link.

  • Creation of a Ground From HeightMap

Parameters are: name, heightmapPath, width, depth, subdivs, minheight, maxheight, scene, updatable

HeightMap grounds are easy, but we decided to create a separate tutorial so we could say more about this important Babylon.js feature. Please see our HeightMap Tutorial to learn all about heightMap grounds.

  • Creation of a Tiled Ground

Thanks to forum user Kostar111 for this handy Tiled Ground constructor. Here is the basic code needed to create a tiled ground.

Parameters are: name, xmin, zmin, xmax, zmax, subdivisions = the number of tiles. (subdivisions.w : in width; subdivisions.h: in height), precision = the number of subdivisions inside a tile. (precision.w : in width; precision.h: in height), scene, updatable.

Kostar111 was also kind enough to give us a fine tutorial about how to use tiled grounds. Click right here to view it. At that link, Kostar111 thoroughly explains how the tiled ground works, and also provides some Babylon.js Playground scenes that nicely demonstrate some of its many uses.

Wrapping Up

And that’s it! Now you have seen all of our basic elements, and some ways to use them. Keep watching this area of the tutorial for new basic elements, as they are being added quite quickly. Feel free to imagine a few of your own basic element ideas, and present them on the forum. Help us make our list of basic elements grow, if you can.
