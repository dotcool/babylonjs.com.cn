      创建一个基本元素(形状或者纹理)

      在这个实例中，我们将要学会如何使用babylongjs创构建一个基本的元素，例如盒子，气缸或者飞机。

      Playground Demo Scene 2 - Seven basic shapes/mesh

      我应该如何开始呢？

      使用基本元素的一个简单方法是访问 Playground Demo Scene 02。你可以下载或者在线的方式浏览整个实例.

      我确信你已经阅读过这篇文章 Babylon.js Primer 和 previous tutorial, 所以你应该知道如何规范化脚本文件，所以我们将不会在这里讨论这个问题，我们将会一步一步得实践这个例子Playground Demo Scene 02。

      让我们来看看这个例子里面的createScene函数:

      我们创建了7个不同的基础元素，并将他们放置到函数的最后边，这是很容易的。我们来谈一谈这7个物体或者纹理的一些特点。

      • 创建一个盒子

      参数分别是名字，大小和关联的场景

      • 创建一个球

      参数分别是名字、段长，大小和关联的场景，段长是用来适应你纹理的大小

      • 创建一个飞机

      参数是名字、大小和场景

      • 创建一个气缸

      参数是名字、高度、距离顶部大小、距离底部大小、镶嵌以及一些可选的参数

      • 创建一个花托

      参数分别是名字，厚度，镶嵌，场景和是否可更新

      • 创建一个结

      参数是名字，半径、管等RIGHT HERE.

      • 一条线的纹理

      参数是名字向量空间和场景
      我可以解释线的纹理是如何工作的，但是我觉得你应该通过运行上边的例子来观察这些语句是如何工作的。

      更多基本的元素 - 地面

      这里我们将要讨论一下这里的基本元素Playground Demo Scene 02, 有3个重要的纹理形状没有被包含在我们的例子中让我们来看一看：

      • 创建一个地面

      参数是名字宽度深度细分值和场景

      我们的第一个例子Playground Demo Scene 01 就是使用了地面这个基本元素

      • 从高度地图创建一个地面

      参数是名字高度地图位置宽度深度细分至最小高度，最大高度，场景是否可更新
      高度地图的地面是很简单的但是我们要把他单独分开详细说明因为他可以展示bablonjs更多的特性。请查看HeightMap Tutorial 学习如何使用高度地图.

      • 创建一个切片地面

      创建一个切片的地面很简单

      参数是名字x最小值，z的最小值，x的最大值，z的最大值，细分值，场景，是否可以更新等
      这里有一个实例是来介绍切片地图的Click right here

      现在你已经查看了我们所有的基础元素以及如何去使用他们，通过学习这部分的例子，我们可以发现很容易就能把创建的新元素添加的界面。
